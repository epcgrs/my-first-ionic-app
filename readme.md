Ionic App
---------


Repositório criado para um curso da Udemy.
quem se interessou é o curso gratis de Ionic.

> **Requisitos:**
> - Node 8.1.2;
> - npm json-server;
> - npm cordova;
> - npm ionic;

in a bash:

1 - run ```npm install ```;

2 - run ```npm i -g ionic ```;

3 - run ```npm i -g cordova ```;

4 - run ```npm i -g json-server ```;

5 - run ```json-server --watch db.json ```;

6 - run ```ionic serve --lab ```;
